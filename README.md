# 20 JavaScript Projects for Beginners

A collection of 20 hands-on JavaScript projects designed to take you from beginner to job-ready frontend developer.

## 🎯 Purpose

These projects are designed for:
- Complete beginners learning JavaScript
- Building a portfolio for entry-level web developer jobs
- Practicing DOM manipulation and event handling
- Understanding core JavaScript concepts through real projects

## 📚 Project List

### Week 1: DOM Basics
1. **Counter App** - Increase/decrease/reset numbers
2. **Color Changer** - Random background colors
3. **Show/Hide Content** - Toggle visibility
4. **Simple Calculator** - Basic math operations
5. **Tip Calculator** - Calculate tips and split bills

### Week 2: Working with Lists
6. **Shopping List** - Add/delete items
7. **Item Counter** - Todo list with counter
8. **Filter List** - Real-time search filter
9. **Quote Generator** - Random quotes
10. **Modal Popup** - Open/close modals

### Week 3: Forms & Validation
11. **Form Validation** - Email validation
12. **Password Strength** - Password meter
13. **Character Counter** - Live character count
14. **Quiz App** - Multiple choice quiz
15. **Survey Form** - Multi-step form

### Week 4: Mini Projects
16. **Countdown Timer** - Countdown from any time
17. **Stopwatch** - Start/stop/reset timer
18. **Image Slider** - Navigate through images
19. **Tabs Component** - Content tabs
20. **Accordion Menu** - Expandable sections

## 🚀 How to Use

1. Clone this repository
2. Open any project folder
3. Open `index.html` in your browser
4. View the code in `script.js` to learn

## 💡 Learning Path

- Spend 1 hour per project
- Complete them in order
- Try to understand every line of code
- Modify and experiment with the code
- Build your own variations

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks)

## 📖 What You'll Learn

- DOM manipulation
- Event listeners
- Functions and variables
- Arrays and objects
- Conditional logic
- Loops and iteration
- Form handling
- Local storage
- CSS styling with JavaScript
